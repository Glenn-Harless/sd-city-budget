# San Diego City Budget: Where the Money Goes

## Overview
Visualize how San Diego collects and spends public money. Compare budgeted vs. actual expenditures and revenue across city departments, track year-over-year trends, and surface spending gaps.

## Key Questions
- How does revenue flow from sources (taxes, fees, grants) into departments?
- Which departments consistently overspend or underspend their budgets?
- How has public safety spending shifted relative to homelessness and infrastructure spending?
- What does spending per capita look like across service areas?

## Available Datasets
- **City Operating Budget vs. Actuals**: https://data.sandiego.gov/datasets/
  - Budget and actual expenditure/revenue data by department and fund
- **Capital Improvement Program (CIP) Budget**: https://data.sandiego.gov/datasets/
  - Capital project budgets and spending

## Data Access
CSV downloads from city open data portal. No API key required.
